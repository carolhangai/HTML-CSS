Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag "b"(não semântica).

Nessa frase, temos um termo em destaque usando a tag "strong"(semântica).

Visualmente as duas tags "b" e "strong" não tem diferenças. Desse modo a tag "b" somente deixa o texto em negrito. Já a tag "strong" traz um significado maior, de modo que esta tag funcione como marcador e "isto que esta em negrito é importante". Se for somente para negritar utilizar "b", se for para enfatizar utilizar "strong".


Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag "i"(não semântico).

Nesta frase, temos um termo em itálico usando a tag "em"(semântica).

A ideia é a mesma do negrito para o itálico.
